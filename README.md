# Model-Lotu-Kuli

Projekt ten ma na celu modelowanie i wizualizację lotu wystrzelonej kuli w czasie.

Wykorzystywane środowisko: Windows Forms

Użytkownik może dostosować takie parametry jak:  
- Prędkość początkowa  
- Masa kuli  
- Opór powietrza  
- Kąt wystrzału
